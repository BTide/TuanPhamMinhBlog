Create Jenkins Pipeline CI process with Jenkinsfile

We can read about jenkinsfile here to get full knowledge about it. In this article, we will learn the way we create a new Jenkins Pipeline project in very basic steps. Therefore, we can understand about it without significant effort. This is the very basic Jenkinsfile in an automation test script. For more details, jenkinsfile is the file that is store in the automation test script.

pipeline {
     agent any

    tools{
        dotnetsdk 'dotnet'
    }

    stages {
        stage('Tests') {
            steps {
                bat "dotnet test"              
            }
        }
    }
}

This Jenkinsfile is the CI process for an Dotnet project. It will define the action execute testing with an bat command line donet test. To run the command dotnet we have to add the dotnetsdk tool. To understand about this, we will move one to the examples.
